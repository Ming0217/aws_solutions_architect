# AWS Solutions Architect — Learning Journal

My personal knowledge base while working through the
[AWS Solutions Architect Learning Plan](https://skillbuilder.aws/learning-plan/EB6SVX4CTK/aws-solutions-architect-learning-plan-includes-labs/SAJSTUCC44)
on AWS Skill Builder, plus any other useful things I pick up along the way.

## How this repo is organized

| Folder | What goes here |
| --- | --- |
| [[learning-plan/learning-plan\|Learning plan]] | Structured notes that follow the official learning plan, organized by topic. Includes the progress tracker. |
| [[labs/labs\|Labs]] | Write-ups of hands-on labs: what I built, gotchas, and screenshots/links. |
| [[projects/projects\|Projects]] | Working AWS-based projects (architecture, tradeoffs, Well-Architected review, walkthrough) — heavier-weight than a lab write-up. |
| [[notes/notes\|Notes]] | Random, ad-hoc learnings that don't fit a specific module. Dated entries. |
| [[resources/resources\|Resources]] | Cheat sheets, useful links, whitepapers, and reference material. |
| `templates/` | Copy-paste templates so notes stay consistent. |

## Quick start

- Starting a new module? Copy [[templates/module-note\|the module template]] into the right `learning-plan/` folder.
- Finished a lab? Copy [[templates/lab-note\|the lab template]] into `labs/`.
- Starting an AWS-based project? Copy [[templates/project-note\|the project template]] into `projects/`.
- Learned something random? Drop a dated entry in [[notes/notes\|notes]] using [[templates/quick-note\|the quick-note template]].
